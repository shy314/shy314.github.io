### while 과 repeat 차이점 

```sw
var i = 0
var sum = 0
while i < 10{
   i += 1
   sum += i
}
print("합계: \(sum)")

var i2 = 0
var sum2 = 0
repeat {
   i2 += 1
   sum2 += i2
} while i2 < 10
print("합계2: \(sum2)")
```

1. 결과 >>> 합계: 55.  //  합계2: 55
2. 결과가 동일하다.

```sw
var i = 10
var sum = 0
while i < 10{
   i += 1
   sum += i
}
print("합계: \(sum)")

var i2 = 10
var sum2 = 0
repeat {
   i2 += 1
   sum2 += i2
} while i2 < 10
print("합계2: \(sum2)")
```

1. i = 10 ,  i2 = 10
2. 결과>>> 합계: 0 // 합계2: 11
3. 결과가 다르다.



결론 : while문은 조건에 만족하면 {}안의 코드 수행한다. repeat문은 코드수행후 조건을 확인한다
